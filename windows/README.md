## 滑动窗口

#### 定义：

```
借助左右双指针维护一个可滑动的窗口；
利用历史计算结果对搜索空间进行剪枝，减少重复计算；
```

#### 2.应用：

```
解决查找满足一定条件的连续区间性质的问题；
例如满足某条件的最x区间（子串、子序列等）；
```

#### 3.算法设计点：

```
1.设计左右指针移动条件；
2.以多重循环思路模拟算法，考虑重复计算特征，抽象为左指针移动条件；
```

#### 4.算法模板

```
初始化最优解；
int l=0; //
for(int r=0;r<nums.length;r++){
	//更新状态
	while(状态满足条件){
		//更新最优解
		l++;
		//更新状态
	}
}
return 最优解
```
